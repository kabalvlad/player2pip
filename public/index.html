Создать комнату

Команда для консоли

1. Открой страницу с видео/аудио
2. Нажми F12 → вкладка Console (Консоль)
3. Вставь команду → нажми Enter
4. Появится ссылка для зрителя — отправь преподу

Как это работает

1
Создай комнату — получи команду
2
Открой страницу с видео/аудио (Skyeng, YouTube и т.д.)
3
Нажми F12 → Console → вставь команду → Enter
4
Отправь ссылку преподу — он увидит и услышит всё, и сможет управлять плеером
